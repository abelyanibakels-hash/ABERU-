# ABERU - Mr Sins Sniper Clips

A gaming clips platform showcasing sniper highlights and gaming content.

## 🚀 Features
- Featured video grid display
- Responsive dark theme design
- Video upload functionality
- User profiles and navigation

## 🛠️ Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. Or run a local server: `python -m http.server 8000`
